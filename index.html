Grid's default to the width of their containing element. Of course, grid's can also have a defined width. Grids are defined using the grid class on a block element.

Row's define vertical gutters, and are defined using the row class.

Columns define horizontal gutters, are are defined using the col class. Column widths are defined as a fraction of the row width using fractional descriptors classes (i.e. full, half, quarter, etc.). So long as the combined width of all columns doesn't exceed the total width of the row, all columns will occupy the same horizontal line.
